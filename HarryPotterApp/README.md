# Harry Potter App - Consumo de API

Aplicativo Android desenvolvido em Java para consumir a [HP-API](https://hp-api.onrender.com/api/) e exibir informações sobre personagens de Harry Potter.

## Funcionalidades

O aplicativo possui 4 telas principais:

### 1. Dashboard (Menu Principal)
- Tela inicial com botões de navegação para todas as funcionalidades
- Botão para sair do aplicativo

### 2. Buscar Personagem por ID
- Campo de entrada para digitar o ID do personagem
- Exibe nome e casa do personagem encontrado
- Validação de entrada e tratamento de erros

### 3. Listar Staff (Funcionários)
- Carregamento automático ao abrir a tela
- Lista todos os membros do staff de Hogwarts
- Exibe nome e ator de cada personagem

### 4. Listar Estudantes por Casa
- Seleção da casa através de RadioButtons:
  - Gryffindor
  - Slytherin
  - Hufflepuff
  - Ravenclaw
- Botão para buscar estudantes da casa selecionada
- Exibe lista completa com nomes e atores
